 **NOTE**
 - Enum is a collection of named integer constant (i.e each element is assigned by integer value)
 - It is declared with enum keyword
 




```

        #include <iostream>
        using namespace std;
        enum Age
           {
                Crystabel=21,
                Prosper=19,
                Iyosayi=18,
                Olaoluwa=23,
                Etinosa=24,
                Nemo=16
            };
            int main()
            {
                enum Age obj;
                obj=Etinosa;
                cout<<"The age of Etinosa="<<obj;
            }


            ### output ###
            The age of Etinosa=24


```
